# Usage
Windows ONLY

Download hnjf（海南鸡饭S+） model [model](https://huggingface.co/chickenrice0721/whisper-large-v2-translate-zh-v0.2-st-ct2)

Go to auto_subtitle.py
change
```
INFER_EXE = Path(
    r"D:\Downloads\hnjf\whisper-large-v2-translate-zh-v0.2-st-ct2-v0.10\infer.exe"
)
```
to your download path.

Drag your video or video folder to bat file.
